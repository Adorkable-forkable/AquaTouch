First Time Setup

- 2 of 8 +

Step 1 - BetterTouchTool

To ensure you have the optimal setup, please set BTT’s [General Touch Bar Settings] as shown below.


Or if you’d like, I can set them for you using the button below.
This will change your current ‘General Touch Bar Settings’ so if you have personally set them up revise them first.
BTT will restart if you press this button.

Automatically Set BTT TouchBar Settings



Step 2 - System Preferences

You’ll need to use the "App Controls" System Touch Bar mode in order to use AquaTouch.
Find this setting in the "Keyboard" section of System Preferences.

Open Keyboard Preferences



The Fn key can be used to toggle between AquaTouch and Stock Touch Bars.
If you would like to switch between AquaTouch and Stock Touch Bars using the Fn key, make sure it is set to "Do Nothing".

On M1 Macs the Fn key is set to switch between Keyboard Languages (A new Globe icon is printed on the key).
You can choose to leave the key as-is and use touch bar buttons to switch to AquaTouch, or, reassign the Fn key to switch Touch Bars instead of toggling the keyboard language.
If you choose the later option, switch the key to "Do Nothing" and use the Menu Bar to switch between keyboard languages.





Minimise Control Strip

- 3 of 8 +

Optional Cleanup

To have a more consistent volume control experience, I recommend minimising your control strip to just one button, then use AquaTouch’s Gestures to control your Volume, Brightness and Keyboard Backlight.
Gestures are easier to use, you'll get more stock touch bar space, and it looks cleaner too!

Click the button below to commit to AquaTouch’s Gesture Controls.
Keep in mind that your current macOS stock ‘Mini Control Strip’ setup will be replaced.

Minimise my Control Strip
If you wish to revert this setting, (bring back stock volume controls), head to [System Preferences] → [Keyboard]






Opening and Closing AquaTouch

- 4 of 8 +

AquaTouch Works with the Stock Touchbar

Unlike other presets, AquaTouch works with the stock touchbar, allowing you to access the (rare and occasional) good controls that some apps give you.
Usually, AquaTouch will switch between it and the stock touchbar automatically, so you won't need to worry about hiding AquaTouch.

Switching between Stock Touchbar and AquaTouch

If you need to switch between presets manually, you can do so via:

Fn Key (Most Recommended)
- Give the Fn key a quick and light tap to toggle between Stock or AquaTouch.
Note: If you seem to also be switching keyboard languages while you do this, check out Page 2, Step 2 of this manual.

TouchBar Buttons
- Switch to Stock: Hold the QuickStrip Apps icon.
- Switch to AquaTouch: Tap the AquaTouch Icon in the stock control strip.





Gesture Controls

- 5 of 8 +

Volume, Brightness and Keyboard Backlight controls in AquaTouch

Here at AquaTouch we have banished akward sliders and touch buttons to control stuff like the volume. You simply can't feel them and they take too long to bring up, find with your eyes, and finally adjust.

With AquaTouch, we use Gestures ; )

How to Adjust

AquaTouch uses Gestures in the TouchBar to control Volume, Brightness and Keyboard Backlight.

Volume
Slide on the touchbar with two fingers to control the Volume

Brightness
Slide on the touchbar with three fingers to control the Screen Brightness

Keyboard Backlight
Slide on the touchbar with four fingers to control the Keyboard Backlight

Note:
Due to limitations in macOS, AquaTouch needs to be open while you slide.
Unfortunately I'm not able to make the gestures work in the stock touchbar, so you'll need to open AQT before doing a gesture. I still find this easier than the stock sliders, even though it takes a bit to get used to.






QuickStrip

- 6 of 8 +

AquaTouch Quick Strip

The Quick Strip is the four-button pallete seen at the very right. It will always be available there for you to access from any aquatouch screen.

It gives you access to helpful widgets, controls and shortcuts that are great to adjust what music you're listening to, adjust some mac settings, check your calendar and more. Here you can read a summary of its four segments.

Quick Media

About
This Segment contains media controls and Now Playing widgets.
If a supported app is playing media, they will show up within this segment.
Additionally for supported apps, media buttons will directly control the playback from that specific app.
e.g. You can pause iTunes, then play Spotify, all from this segment, without touching the mouse.

Hold Action
You can hold this segment button to Play/Pause Media.

Icon Color
The icon for this segment will turn blue if media is playing.

Quick Schedules

About
Tap to check your calendar and reminders. You can also view the weather and time here.

Hold Action
Hold this control strip button to toggle Do Not Disturb

Quick Controls

About
Set system controls such as NightShift, Dark Mode or to sleep your MacBook. There's a connectivity pallete to the right that allows you to connect to AirPods, Beats, toggle WiFi, toggle Do Not Disturb and more.

Hold Action
Hold this control strip button to Open a Bluetooth Tool

Icon Color
The icon for this segment will turn violet if Do Not Disturb is active.

Quick Apps

About
Tap this segment to launch a quick task with one button. You can hold command here to quit multiple apps with a touch, too.

Hold Action
Hold this segment button to switch to the stock MacOS Touch Bar.




Web Tools

- 7 of 8 +

Web Tools

AquaTouch supports many apps, and it even supports websites!

If you visit a supported website in Safari or Google Chrome, you'll notice an additional button is added to the AquaTouch Controls.
Simply tap this icon to bring up website-specific controls.

Currently Supported Websites:

  • YouTube
  • Twitch
  • SoundCloud
  • Gmail
  • Google Drive
  • Google Docs
  • Google Sheets
  • Google Slides
  • Facebook
  • Kahoot.it
  • Netflix (Safari only for now)

    • Setting up YouTube and Netflix

    Youtube and Netflix Web Tools have deeper integration with Safari and Google Chrome.
    You'll need to allow them to access the web browser for them to work properly.

    Allowing Javascript Access

    Safari
    Head to Safari Preferences [ ⌘ , ]
    Hit the Advanced Tab
    Check "Show Develop Menu in Menu bar"

    Then, in the menu bar, hit [Develop → Allow JavaScript from Apple Events]

    Google Chrome
    In the menu bar, hit [View → Developer → Allow JavaScript from Apple Events]




    - 8 of 8 +

    Permission Popups & Privacy


    As you use AquaTouch for the first time, these popups may frequently appear for many apps.


    AquaTouch works by interating with on-screen menus and system controls, this is why these popups appear.
    In order for AquaTouch to work properly, Allow BetterTouchTool to interact with these items.


    Privacy

    AquaTouch does not broadcast any data anywhere. Simple as that.
    BetterTouchTool may send anonymous analytical or crash/debug data to the developer in order to improve it, check BetterTouchTool's privacy statement for it's stand.

    Microphone
    You can mute your microphone or toggle Active Noise Cancellation for some earbuds with AquaTouch. You can also adjust speaker volume with AquaTouch.

    Contacts, Calendar
    AquaTouch uses BetterTouchTool to read your Calendar and Reminders to show your Schedules in the touchbar. If you decide to make a new event from the touchbar, you'll have access to your contacts too.

    Files
    AquaTouch can create new files for you (especially in finder) and show you connected external disks. It also has shortcuts to folders such as Desktop, Downloads for you to access with a tap.

    Screen Recording
    AquaTouch reads menu items and clicks them as you use the touchbar.

    Location
    BetterTouchTool may read your approximate location to adjust AquaTouch's Clock and Weather features to your local area.

    Automation and Accessibility API
    AquaTouch uses the macOS Accessibility API to interact with menus (e.g. reading the menubar and clicking items in it), or directly with apps (e.g. to play/pause youtube or music)

    Javascript from Apple Events (web browsers)
    AquaTouch uses Javascript from Apple Events to read and control websites, such as the pause button, video name and playback position in youtube, or to detect a video and show the picture-in-picture button.
